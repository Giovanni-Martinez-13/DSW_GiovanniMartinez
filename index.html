CSS3: Estilos internos y Externos, Selectores CSS


Este sitio web se desarrolla con el fin de poner en practica situaciones reales del mundo laboral las cuales serviran como guia y puesta en marcha de los conocimientos basados en la materia de Diseño de sitios Web.

Objetivo general:
Realizar un OVI utilizando metodologías HTML5 Y CSS3 para poner en práctica los conocimientos adquiridos en el curso y de esta manera obtener habilidades para el futuro laboral.
Objetivo específicos
Explicar temas relacionados con CSS3: Estilos internos y Externos, Selectores CSS.
Dar a conocer framework utilizados para facilitar el diseño profesional de sitios web
Explicar los pasos para la implementación de una hoja de estilos utilizando videos y o contenido interactivo.
